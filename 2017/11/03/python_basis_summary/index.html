条件语句

if x > 0:
    print("x>0")

如果什么都不执行

if x < 0:
    pass

二选一

if x > 0:
    print("x>0")
else:
    print("x<=0")

链式条件

if x < y:
    print("x<y")
elif x > y:
    print("x>y")
else:
    print("x==y")

嵌套条件

if x == y:
    print("x==y")
else:
    if x < y:
        print("x<y")
    else:
        print("x>y")

其他特性

if 0 < x < 10:
    print("0<x<10")

字符串操作

字符串是由字符组成的序列。你可以用括号运算符一次访问一个字符：

>>> s='abcd'
>>> s[1]
'b'

字符串的长度

>>> s = 'abcd'
>>> len(s)
4

while 循环遍历字符串

s='abcd'
index = 0
while index < len(s):
    letter = s[index]
    print(letter)
    index = index + 1

for 循环遍历字符串

s='abcd'
for letter in s:
    print(letter)

字符串切片

字符串的一个片段被称作切片 (slice)。选择一个切片的操作类似于选择一个字符：

s='abcdefghijk'
print(s[0:5])  # abcde
print(s[2:5])  # cde

操作符 [n:m] 返回从第 n 个字符到第 m 个字符的字符串片段，包括第一个，但是不包括最后一个。

或者可以理解为从第 n 个字符开始，截取 m-n 个字符

可以省略其中一边的索引

>>> s = 'abcdef'
>>> s[:3]
'abc'
>>> s[3:]
'def'

如果第一个索引大于或等于第二个，结果是空字符串 3 ，用两个引号表示：

>>> s = 'abcdef'
>>> s[3:3]
''

一个空字符串不包括字符而且长度为 0，但除此之外，它和其它任何字符串一样。

python 中的字符串同样是不可变的，不要试图去改变一个字符串。

s='abcdefghijk'
print('f' in s)  # True
print('mm' in s)  # False
