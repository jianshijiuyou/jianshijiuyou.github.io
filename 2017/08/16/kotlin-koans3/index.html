第二十一经 Introduction

在 kotlin 中默认使用的是 java 集合，但是 kotlin 提供了很多扩展方法，比如不同集合之间的转换，这些方法都以 to 开头，比如 toSet 或 toList：

val list= listOf("a","b","c")
var set=list.toSet()

第二十二经 Filter map

Filter： 过滤所有符合给定函数条件的元素。
map： 返回一个每一个元素根据给定的函数转换所组成的List。

val numbers = listOf(1, -1, 2)
numbers.filter { it > 0 } == listOf(1, 2)
numbers.map { it * it } == listOf(1, 1, 4)

第二十三经 all, any, count, find

all： 如果全部的元素符合给出的判断条件，则返回true。
any：如果至少有一个元素符合给出的判断条件，则返回true。
count：返回符合给出判断条件的元素总数。
find：返回第一个符合判断条件的元素，没有则返回 null。

val numbers = listOf(-1, 0, 2)

val isZero: (Int) -> Boolean = { it == 0 }

numbers.any(isZero) == true

numbers.all(isZero) == false

numbers.count(isZero) == 1

numbers.find { it > 0 } == 2

第二十四经 FlatMap

遍历所有的元素，为每一个创建一个集合，最后把所有的集合放在一个集合中。

val result = listOf("abc", "12").flatMap { it.toCharList() }

result == listOf('a', 'b', 'c', '1', '2')

第二十五经 max, min, maxBy, or minBy

max：返回最大的一项，如果没有则返回null。
min：返回最小的一项，如果没有则返回null。
maxBy：根据给定的函数返回最大的一项，如果没有则返回null。
minBy：根据给定的函数返回最小的一项，如果没有则返回null。 

val list = listOf(1, 2, 3, 4, 5, 6)

assertEquals(6, list.max())

assertEquals(1, list.maxBy { -it })

assertEquals(1, list.min())

assertEquals(6, list.minBy { -it })

第二十六经 Sort

sort：返回一个自然排序后的list。
sortBy：返回一个根据指定函数排序后的list。

listOf("bbb", "a", "cc").sorted() == listOf("a", "bbb", "cc")

listOf("bbb", "a", "cc").sortedBy { it.length } == listOf("a", "cc", "bbb")

第二十七经 Sum

sum：返回所有元素的总和，支持 Byte , Short , Int , Long , Float , Double。
sumBy：返回所有每一项通过函数转换之后的数据的总和，支持 Int。
sumByDouble：同上，支持 Double。

listOf(1, 5, 3).sum() == 9

listOf("a", "b", "cc").sumBy { it.length() } == 4

第二十八经 Group By

返回一个根据给定函数分组后的 map。
分组的条件就是 map 的 key，比如这里分组的条件是字符的长度，那长度就是 key。

val result = listOf("a", "b", "ba", "ccc", "ad").groupBy { it.length() }

result == mapOf(1 to listOf("a", "b"), 2 to listOf("ba", "ad"), 3 to listOf("ccc"))

第二十九经 Partition

把一个给定的集合分割成两个，第一个集合是由原集合每一项元素匹配给定函数条件返回true的元素组成，第二个集合是由原集合每一项元素匹配给定函数条件返回false的元素组成。

val numbers = listOf(1, 3, -4, 2, -11)

val (positive, negative) = numbers.partition { it > 0 }

positive == listOf(1, 3, 2)

negative == listOf(-4, -11)

第三十经 Fold

在一个初始值的基础上从第一项到最后一项通过一个函数累计所有的元素。

listOf(1, 2, 3, 4).fold(1, {

partProduct, element ->

element * partProduct

}) == 24

第三十一经 | 第三十二经 Get used to new style

public Collection<String> doSomethingStrangeWithCollection(Collection<String> collection) {

    Map<Integer, List<String>> groupsByLength = Maps.newHashMap();

    for (String s : collection) {

        List<String> strings = groupsByLength.get(s.length());

        if (strings == null) {

            strings = Lists.newArrayList();

            groupsByLength.put(s.length(), strings);

        }
        strings.add(s);
    }

    int maximumSizeOfGroup = 0;

    for (List<String> group : groupsByLength.values()) {
        if (group.size() > maximumSizeOfGroup) {
            maximumSizeOfGroup = group.size();
        }
    }

    for (List<String> group : groupsByLength.values()) {

        if (group.size() == maximumSizeOfGroup) {
            return group;
        }
    }
    return null;
}

将上面这段 java 代码用 kotlin 风格重新编写：

fun doSomethingStrangeWithCollection(collection: Collection<String>): Collection<String>? {

    val groupsByLength = collection. groupBy { s -> s.length }

    val maximumSizeOfGroup = groupsByLength.values.map { group -> group.size }.max()

    return groupsByLength.values.firstOrNull { group -> group.size==maximumSizeOfGroup }
}
